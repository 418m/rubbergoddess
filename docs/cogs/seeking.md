# Seeking

Announce that you're seeking something.

## User commands

### seeking

Display items for current channel.

### seeking add (text)

Add item. The text has to be 140 characters long at most.

### seeking remove (id)

Remove item.

## Privileged commands

This module has no commands only usable by privileged users.
